# 🚐 VanTrack - Monitoramento de Vans

Projeto de monitoramento de vans para estudantes universitários.

## 📱 Funcionalidades
- Localização em tempo real
- Tempo estimado
- Rotas

## 🎯 Objetivo
Melhorar a organização e segurança no transporte de estudantes.
